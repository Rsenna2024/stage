Ik had nog nooit met laravel gewerkt. Om aan de slag te gaan heb ik de videos van Dani Krossing bekeken. Tijdens het maken van dit project heb ik de videos als guidline gebruikt. 

Aan Co-pilot heb ik de volgende dingen gevraagd. 
"Kan je me vertellen waar ik het beste kan beginnen?" Dit heb ik gevraagd, omdat ik niet een heel project met codes wou hebben, maar ik vondt het lastig om de handigste volgorde te bepalen.

Ook heb ik gevraagd, waarom ik een 404 page not found fout code had. Als eerste had ik de foutcode gegoogle, maar daar kwam ik niks verder mee. Toen had ik de hulp van Co-Pilot, maar daar kwam ik ook niet uit. Dus ik ben opnieuw begonnen om te kijken of het tijdens het opzetten van het document fout is gegaan. Na alles verwijderen en opnieuw rustig nadenken waarom doe ik wat lukte het wel. Daarna heb ik het project gelijk goed opgeslagen zodat ik als het fout ging, een stapje terug kon doen naar werkende codes.
